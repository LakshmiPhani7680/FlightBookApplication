Acceptance criteria <br>

1. RESTful Architecture <br>
   ---Develop a RESTful API using Node.js, ExpressJS, and MongoDB for efficient data handling and routing control.<br>
2. Code Modularity <br>
   ---Organize code using ES6 Modules for maintainability and modularity.
   ---Secure routes so only authenticated users can book, view, or cancel.
3. <b>User Authentication</b>
<ul>
<li>Implement a user authentication system with essential features like register, login, and logout.</li>
<li>Registration should include user details such as name, email, password, role, profilePicture, createdAt and gender.</li></ul></pre>
